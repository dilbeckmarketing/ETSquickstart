How to add designations to agent profile pages

Learn how to add Designations to Agent Profile pages with our "Agent Highlight" feature. Detailed instructions also below.

WATCh video


The Designation is added to the “Agent Highlights” section of their profile in the CRM. Entering HTML code is required to have the heading and text match the other titles, texts, and features on the page.


Here is the Designation Code you need to enter:

<div id="team_offices">
<div class="body-semi-title">Designations</div>
<div class="smidge">&nbsp;</div>
<p>MVSC</p>
</div>

Here's how to enter it:

Navigate to the agent profile in an agent CRM by clicking on their headshot in the top right corner. Select “My Profile”
￼

Click “Edit”

Scroll down to “Agent Highlights”

Click on “Source”

In the text box, paste provided Designation Code

Then click “Update Agent”

Designation section will now be visible on the agent profile page in the agent highlight section, to the right of the bio.

Adding Additional Designations:

Return to editing the Agent Profile and scroll down to "Agent Highlight" section.

Click after the last letter of the first designation and hold the “SHIFT" key and press the "RETURN” key to go to the next line. 

Once on the next line, type out your designation.

You can also link out to NAR detail pages for the designation by highlighting the designation and selecting the hyperlink option. 

Paste destination URL in pop-up window.
