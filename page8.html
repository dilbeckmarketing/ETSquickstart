Using Email Templates

Email Marketing – Test, Clone and Add Dynamic Content

Test Newsletters or Drip Campaigns

In order to test newsletters, go to Marketing > Campaigns > Drip Campaigns or Marketing > Campaigns > Single Email Events. On this page you will have the ability to send a test email for any of your unsent Newsletters or any Drip Campaign by clicking the “Test” button (paper plane icon) in the far right of the individual campaigns/newsletters.

When the test button is clicked you can set a lead that will fill in the dynamic information as well as any additional emails that you would like to copy on the test email. (By default the email associated with the account logged in will be emailed the test.)


Adding Dynamic Content to a Template

How to Clone a Template

An already created template may be cloned, so it can be updated and retain similar formatting to the previous template.

The clone can be created by going to Email Marketing > Templates and clicking the “Clone” (two squares icon) in the far right of any existing template.

Once “Clone” has been clicked, you will be taken to the WYSIWYG editor for the template. Here you can rename the template and make any alterations that you would like to make to the cloned template.

Creating a Template

1. Go to Marketing > Templates.

2. Click “Create Template.”

3. Name the Template and add a subject

• The subject will be the default subject of the received email, if no changes made when creating a newsletter/drip campaign.

• Brokers will have the option to create a company or personal template and can limit the access to Administrative accounts.

4. Select one of the Template layouts

• Some templates layouts will require additional information to be entered on this page

5. Click the Save button

6. Make any edits to the formatting of the template in the wysiwyg editor.

7. Click “Save Template.”
