Homepage Customization

Change Homepage Image

Path: Content > Theme Photos > New Theme Photo 

A theme photo or video can be added to the front end of your homepage.

On the New Theme Photo page, click Upload Image to choose an image. 

You can also upload a video. The video does NOT play audio and must be an MP4(H.264/AAC) and less than 21 seconds long, and less than 8mb.

Once the image/video is chosen, you have the option to enter a Caption (this text will be in the middle of your theme photo) 

You can also link your photo to a page on your site, or an external link. Choose the local path from the Linked Page dropdown or enter the url for the external link in the Page Path field. 

The Active checkbox is turned on by default for new theme photos. This enables the photo to be visible on the front end. If you would like the photo to not display in the front end, but still would like it to be there on the back end, simply edit the photo of choice and uncheck the Active checkbox.

Customize Site Menu

Select the Content tab on the back end of your website and then click the “Menus” sub-item to begin. Here you will be able to modify the primary navigation of your site, as well as any desired navigation in the header and footer of all pages. You can also create custom sidebar menus for use on any page you want. If you are using Search Pages, you can also create a special global navigation menu for your Search Pages which will appear just above the footer on all pages of your website.

To Add Items to a Menu

Choose your menu: Primary Navigation, Header Navigation, Footer Navigation, Town Pages, or any custom menus that you have created

Choose a main menu item or “Add s Sub Item”

Enter a title for the menu item, and either select the page you want to link to from the dropdown list, or enter an external URL in the External Link field

Reorder any main menu item or sub menu item by grabbing and dragging the “Reorder Tool”

Click “Create” and your new menu item will be added to the selected menu

To Edit a Menu Item

Choose your menu: Primary Navigation, Header Navigation, Footer Navigation, Town Pages, or any custom menu that you have created

Simply edit the text, choose the page you want from the dropdown list or add an external link.

To Delete a Menu Items

Choose your menu: Primary Navigation, Header Navigation, Footer Navigation, Town Pages, or any custom menu that you have created

Click the “Delete” button for a main menu item or click the red “x” to delete the sub menu item.

To Create a Sidebar Menu

Select the Content tab on the back end of your website, then click the “Menus” sub-item

Click “Create New” in the upper right of the Menus editor

Give you new custom menu a name and description

Click “Create” to save your changes

Add any desired menu items using the same process as described in the “To Add Items To A Menu” section of this guide

To Add a Sidebar Menu to a Page

Select the Content tab on the back end of your website; the “Pages” sub-item opens by default

Choose which page you wish to add a menu to and click the edit tool 2013-08-12_15-17-30

Choose which of your sidebar menus you wish to add to the page by using the “Sidebar Menu”dropdown located just above the sidebar widgets.

Click “Update” to save your changes

**Note: A sidebar menu can be applied to as many pages as you want. Only one sidebar menu can be applied to each page.

Tip: You can create navigation links to any page, from any page, using the Content feature. For example, if you have a number of pages related to one concept – such as Search or neighborhood pages – you can create a text or image link on each item’s page that returns the user to the main page you have created for the category.

Homepage Callouts

Callouts can be created to show on the homepage. The look of the callouts depends on the theme. Once created, you are able to link the callouts to the content/search pages of choice. 

To create a homepage callout

Content > Homepage Callouts > New Homepage Callout

Add the picture for the callout, enter a heading/caption, choose a local page to link the callout to or enter an external link. 

The active check box is turned on by default. Uncheck the box to deactivate the callout. 

Continue to create up to 6 callouts > you can Edit/Delete/Reorder the callouts as preferred. 

Activate the callouts on homepage

Settings > Configurations > Company > under Homepage Options, choose the Display Option and the Number of Callouts > Save Settings

The callouts will populate on the homepage as configured.
